运行python

windows

你可以轻松的下载windows版的python,我推荐下载python2而并非python3（别问我为什么，python3开发远远比普及要快）

下载完成后，别忘了把python.exe的路径加到环境变量里面。

为了更方便的安装python的包，推荐安装setuptools，下载这个脚本，然后再命令行中运行就可以了。

python本身的终端并不是很好用，都没有自动补全。我推荐使用ipython,之前已经安装过了setuptools,在命令行中运行。

pip install ipython

就可以安装ipython了,如果过程中提示缺少什么模块，可以再用pip安装。

linux

linux肯定是自带python的，所以问题就是安装ipython就可以了

sudo apt-get install ipython

或者是redhat系的

yum install ipython

都能很方便的安装。

除此之外

除此之外，一个好的编辑器也是可以帮助你写python的利器，如vim，Sublime，或者是Eclipse+pydev都可以。

到现在你应该完成

  • 有一个python的shell环境，推荐用ipython。
  • 一个用来写代码的环境，python以及pip并且在path中。
  • 一个可以方便上网的环境。

现在你可以打开你的python shell，准备进入下面的旅程了。

python的交互界面是很典型的REPL(Read-Eval-Print Loop)环境，输入一段代码，回车就会执行这部分代码，然后给出返回值。一般一行是一个表达式，但是你也可以在一个表达式后面加上';'，然后再写另一个表达式。

In [4]: print 'hehe';print 'hehe'
hehe
hehe

print是我们见到的第一个语句，它会打印出之后的内容，并且在后面加上一个'\n'，如果你不想让它换行的话在后面的内容后加一个','就可以了。

In [5]: print 'hehe',;print 'hehe'
hehe hehe

变量

python可能不同于你之前学过得其他语言，它的变量是不用声明的，你可以随意的直接赋值，而不用声明它是什么类型的。

In [14]: a = 1
In [15]: b = "hehe"
In [16]: print a 
1
In [17]: print b
hehe

谈到变量，大家肯定会想到整型，字符串神马的，用python的时候基本不用注意这些，python的数很大，达到我们认为是无穷大，所以你不用担心整型越界的问题，字符串就是夹在""或者是''中的内容，有一些特殊的用法会在之后提及。

这里还是要提几句，如果java是一切都是类的话，python应该是一切都是对象，python中没有基本类型这种东西，所有的东西都是对象，这种事不懂也无所谓，我就是提一句。

有个函数type可以返回某个对象是什么类型的

In [31]: type(12.123)
Out[31]: float
In [32]: type("hehe")
Out[32]: str

python的运算符也是重载过得，（如果你连重载是什么意思都不记得了就当我没说）,想加号可以用来加数（废话）也可以用来连接其他东西比如字符串、列表（还没讲到不用管它）

其实最简单的一句话，你感觉应该有的操作，python都有，还有*是乘号，**（两个乘号）就是乘方的意思了，是不是很方便。

In [35]: 'fuck you three times' + ' everyday'
Out[35]: 'fuck you three times everyday'
In [36]: 2**3
Out[36]: 8

带有基本栅格的HTML代码

对于简单的两列式布局，创建一个 .row 容器，并在容器中加入合适数量的 .span* 列即可。由于默认是12列的栅格，所有 .span* 列所跨越的栅格数之和最多是12(或者等于其父容器的栅格数)。

  1. <div class="row">
  2.   <div class="span4">...</div>
  3.   <div class="span8">...</div>
  4. </div>

上面的代码展示了 .span4 和 .span8 两列，两列的和总共是12个栅格。

偏移列

把列向右移动可使用 .offset* 类。每个类都给列的左边距增加了指定单位的列。例如，.offset4 将 .span4 右移了4个列的宽度。

3 offset 1
3 offset 2
6 offset 3
  1. <div class="row">
  2.   <div class="span4">...</div>
  3.   <div class="span3 offset2">...</div>
  4. </div>

嵌套列

在默认的栅格系统里, 在已有的.span*内添加一个新的 .row 并加入 .span* 列，就可实现嵌套。被嵌套列中的每列列数总和要等于父级列。

布局

固定布局

提供了一个通用的固定宽度(也可以变为响应式)的布局方式，仅仅用 <div class="container"> 即可。

  1. <body>
  2.   <div class="container">
  3.     ...
  4.   </div>
  5. </body>

流式布局

利用 <div class="container-fluid"> 代码可以创建一个流式、两列的页面 — 非常适合于应用和文档类页面。

  2.   <div class="row-fluid">
  3.     <div class="span2">

  5.     </div>
  6.     <div class="span10">

响应式设计

启用响应式特性

通过在文档中的 <head> 标签里添加合适的meta标签并引入一个额外的样式表即可启用响应式CSS。如果你已经在定制页面编译好一个Bootstrap, 那么只需添加一个meta标签。

  1. <meta name="viewport" content="width=device-width, initial-scale=1.0">
  2. <link href="assets/css/bootstrap-responsive.css" rel="stylesheet">

注意！ Bootstrap在默认情况下是没有引入响应式特性的，因为不是任何情况都需要使用到。我们并不是鼓励开发者移除此功能, 而是在需要使用的时候才启用它。

关于响应式Bootstrap

Media queries允许在一些条件基础上自定义CSS 例如：—ratios、widths、display type, etc— 但通常都是围绕着 min-width 和 max-width 这两者。

  • 修改栅格系统中列的宽度
  • 需要的时候, 用堆叠元素代替浮动
  • 调整标题和文本的大小以便适合各种设备

谨慎的使用media queries，先从你的手机用户开始吧。对于大型的项目, 应该考虑使用专门的代码库, 而不是构筑在media queries之上。

被支持的设备

Bootstrap 所支持的几个media queries都放在了一个文件中， 可以让你的项目更灵活的去适应不同设备和屏幕分辨率。包括：

类型 布局宽度 列宽 间隙宽度
大屏幕 大于等于1200px 70px 30px
默认 大于等于980px 60px 20px
平板 大于等于768px 42px 20px
手机到平板 小于等于767px 流式列，无固定宽度
手机 小于等于480px 流式列，无固定宽度 
  1. /* 大屏幕 */
  2. @media (min-width: 1200px) { ... }
  3.  
  4. /* 平板电脑和小屏电脑之间的分辨率 */
  5. @media (min-width: 768px) and (max-width: 979px) { ... }
  6.  
  7. /* 横向放置的手机和竖向放置的平板之间的分辨率 */
  8. @media (max-width: 767px) { ... }
  9.  
  10. /* 横向放置的手机及分辨率更小的设备 */
  11. @media (max-width: 480px) { ... }

何时使用

在有限的情况下使用, 避免创建同一个站点的不同版本。当这些class能对每种设备的展示做有益的补充的时候才使用。响应式工具不能用于table元素，而且本就不支持。
